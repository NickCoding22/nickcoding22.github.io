Projects:

  • BotChecker [Twitter Bot]

    A Twitter bot developed with Twitter4j / the Twitter API and the Botometer API to determine a confidence interval surrounding the proportion of active Twitter accounts that are automated. It was run on a Raspberry Pi 4 for each hour of every day for seven days to obtain comprehensive data of a full week of activity.

  • Dartboard Investing [Stock Investment]

    An unorthodox stock investment tool: through the Alpaca stock trading API and a Raspberry Pi 4, an electric dartboard was rigged to automatically invest in the stocks chosen by a dart landing in a spot corresponding to a given stock. 22 fractional stocks were randomly chosen from the NASDAQ each cycle and assigned spots on the board with varying investment levels.
